# WaterBox App

## Executando o projeto

### Metodo - Executável (Somente em Windows)

- Clicar no arquivo referente o seu Os que a aplicação será executada.
  - Para Linux -> `waterbox-linux`
  - Para Windows -> `waterbox-win.exe`
